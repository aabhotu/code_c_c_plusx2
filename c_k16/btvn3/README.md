# Bài 5: (20đ)

Viết chương trình: Nhập số tự nhiên n. Nếu n >=0 và n<=100 thì tính và in ra màn hình giá trị các biểu thức sau đây:\
A = 1 + 2 + 3 + … + n; \
B = 1 + 2 + 4 + … + 2*n;\
C = 1 + 3 + 5 + … + (2*n + 1);\
**Note: Sorry đề chưa được chặt chẽ, nên nếu n == 0 thì A = 0; B = 0; C = 1.**
